# スプレッドシートの項目（列）について

マスターとなるスプレッドシートの列は、**追加したり並び順を変えたりするのは自由**です。一方で、**サイトが表示に使っている項目名（キー）** を変えると、その部分が表示されなくなるか、エラーになることがあります。

---

## 項目を「少し変更」してもよいか？

| 変更の種類 | 問題ないか | 補足 |
|------------|------------|------|
| **列を追加する** | ✅ 問題なし | サイトは使わない列は無視します。メモ用・担当者名など自由に追加してよいです。 |
| **列の並び順を変える** | ✅ 問題なし | サイトは列名（キー）で参照するため、順序は関係ありません。 |
| **使わない列を削除する** | ✅ 基本的に問題なし | その項目がサイトに表示されなくなるだけです。**group_id** だけは「どのグループか」の識別に使うため、残すことを推奨します。 |
| **列名（1行目の項目名）を変える** | ⚠️ 注意 | スプレッドシートの**1行目＝JSON のキー名**になります。サイトのコードは下記の「サイトが参照するキー」を読んでいるため、**キー名を変えるとその項目が表示されなくなります**。 |

---

## サイトが参照しているキー（ここを変えると表示に影響する）

スプレッドシートの**1行目（ヘッダー）**を次の名前にしておくと、そのままサイトで使われます。**日本語の列名にしたい**などで変える場合は、**エクスポート後に JSON のキーをこの名前に揃える**か、サイト側のコードを同じ名前に合わせて修正する必要があります。

### 一覧用（トップのスケジュール表）で使うキー

| キー名 | 説明 |
|--------|------|
| `group_id` | グループ識別子（例: 001, 201-1）。詳細ページへのリンクに使う。**必須**。 |
| `group_name` | グループ名（表に表示） |
| `theme_title` | 探究テーマ／発表タイトル（表に表示） |
| `timeslot_label` | 時間帯（例: 10:00-10:10） |
| `room_name` | 教室名 |

### グループ詳細ページで使うキー

| キー名 | 説明 |
|--------|------|
| `theme_detail` | 発表概要 |
| `pdf_title` | PDF の表示名 |
| `pdf_drive_url` | PDF を「新規タブで開く」用の URL |
| `pdf_embed_url` | PDF の埋め込み表示用 URL |
| `slides_title` | 発表スライドの表示名（任意） |
| `slides_pdf_drive_url` | **聴衆向け** 発表スライド PDF を「新規タブで開く」用の URL（バックアップ兼用） |
| `slides_pdf_embed_url` | **聴衆向け** 発表スライド PDF の埋め込み表示用 URL |
| `slides_present_url` | **発表者用** Google スライドの **プレゼンモード** URL（投影・アニメーション用） |
| `slides_view_url` | **発表者用** Google スライドの閲覧用 URL（任意） |

※ **聴衆向け**（グループ詳細ページ）は PDF のまま。**発表者用ページ**では Google スライド（`slides_present_url`）を優先してプレゼンモードで開きます。機材不調時は聴衆向け PDF がバックアップになります。

### JSON の先頭で使うキー（1行目ではなく別管理でも可）

| キー名 | 説明 |
|--------|------|
| `eventTitle` | イベント名（通常は固定で OK） |
| `eventDate` | 開催日（例: 2025年〇月〇日） |
| `notice` | 注意書き |

---

## 運用のコツ

- **列名は英語のキー名のまま**にしておくと、エクスポートした JSON をそのままサイトに渡せて確実です。
- **列の見出しを分かりやすくしたい**場合は、スプレッドシートの「2行目に日本語の説明」を置き、1行目はキー名（上記の英語）のままにする方法があります。
- 項目を**追加**する分には制限はありません（`担当者`, `備考2` など）。

要約すると、**「追加・並び替え・不要列の削除」は自由で問題ありません。列名（キー）を変えるときだけ、JSON のキーとサイトの表示が一致するように注意**すれば大丈夫です。
